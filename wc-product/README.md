# wc-product

Micro-frontend that builds as a Web Component.

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Compiles to Web Component and minifies for production

```
npm run build:wc
```

### Lints and fixes files

```
npm run lint
```
